# Fluffball - Cloud Bunny

This is an example pet directory structure.

## Contents

- `pet_info.json` - Complete pet data (personality, appearance, actions, metadata)
- `image.png` - Base character image (to be generated)
- `animations/` - Directory containing sprite animations for each action
  - `idle.gif`
  - `hop.gif`
  - `float.gif`
  - etc.

## Pet Information

**Species:** Cloud Bunny
**Personality:** Playful, curious, gentle, bouncy
**Special Ability:** Can float on air currents and create small rainbow trails

## Actions

1. idle - Gentle floating bobbing motion
2. hop - Bouncy hopping movement
3. float - Graceful floating through the air
4. wiggle_ears - Ear wiggling expression
5. bounce - Energetic bouncing
6. sleep - Curled up sleeping on a cloud
7. curious_look - Looking around with wide eyes
