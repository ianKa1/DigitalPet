# Generated Pets Directory

This directory contains all generated digital pets. Each pet has its own subdirectory with the following structure:

```
pet_name/
├── pet_info.json       # Complete pet data (personality, appearance, actions)
├── image.png           # Base character image
├── animations/         # Sprite animations for each action
│   ├── idle.gif
│   ├── walk.gif
│   └── ...
└── README.md           # Pet description and info
```

## Pet Naming

Pet directories are named using the pet's name in lowercase with underscores, e.g., "Fluffball" → `fluffball/`
